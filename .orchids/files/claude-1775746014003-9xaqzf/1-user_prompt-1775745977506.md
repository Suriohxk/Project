ROLE:
You are a senior full-stack AI engineer and product architect.

GOAL:
Build a complete AI-powered "Waste Classification and Smart Disposal Assistant" system aligned with SDG 12 (Responsible Consumption and Production), specifically designed for Indian users.

CONTEXT:
The system solves the problem of improper waste segregation in Indian households by providing real-time AI-based waste classification and disposal guidance.

SYSTEM OVERVIEW:
- Accept natural language input (e.g., "plastic bottle", "banana peel", "battery")
- Optionally accept image input for waste classification
- Classify waste into categories:
  - Plastic
  - Organic
  - E-waste
  - Hazardous
- Provide simple, step-by-step disposal instructions based on Indian practices
- Collect user details (name, age, query)
- Use polite, eco-friendly conversational flow
- Trigger webhook when all data is collected

---

BACKEND ARCHITECTURE (LangGraph):

Nodes:
1. Start Node:
   - Accept user input

2. Router Node:
   - Classify waste type using simple LLM or rules

3. Waste Nodes:
   - Plastic Node → disposal steps
   - Organic Node → compost/green bin
   - E-Waste Node → recycling center guidance
   - Hazardous Node → safe disposal instructions

4. Clarification Node:
   - Ask one question at a time
   - Collect:
     - user_name
     - user_age
     - waste_item

5. Final Node:
   - Generate response
   - Trigger webhook

---

CLASSIFICATION LOGIC:
- Use keyword-based + LLM fallback
- Example:
  - "banana peel" → Organic
  - "battery" → Hazardous
  - "plastic bottle" → Plastic

---

DISPOSAL LOGIC (INDIA-SPECIFIC):
- Plastic → Dry waste / recycling bin
- Organic → Wet waste / compost
- E-waste → Authorized e-waste centers
- Hazardous → Special disposal (batteries, chemicals)
